Wordpress Scroll To Top plugin does the job with just one click a button appears when a person scrolls to the middle of the page to be clicked on.


== Description ==
Wp Scroll to top plugin makes the scrolling of a page easier. It appends an icon at bottom of the page which attracts user attention. When the user click on this button it smoothly scrolls to the top of Webpage. Wp Scroll To Top plugin gives you custom settings which no other plugin gives.


* Only Wordpress plugin tool for scroll to top.
* Support will be provided.
* Fluid/responsive icon.
* Compatible with all Wordpress themes.
* It is user friendly and very easy to install.
* Zero coding skills required to set up this plugin.

 == Installation ==
* Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.



== Frequently Asked Questions == 
Q.How to change the background color ?

Ans: Open the scr_style.css file  and change the background-color  field.
